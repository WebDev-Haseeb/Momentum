# Momentum

Momentum is a mobile-first personal productivity app for daily focus. It combines tasks, habits, notes, settings, and local-first persistence into a compact app-like experience.

## Tech Stack

- React 19
- TanStack Start and TanStack Router
- TypeScript
- Vite
- Tailwind CSS
- Zustand
- IndexedDB
- Nitro for Vercel deployment

## Getting Started

```sh
npm ci
npm run dev
```

The dev server runs on port `8080` by default.

## Useful Commands

```sh
npm run lint
npm run format
npm run build
```

## Deployment

This project is configured for Vercel with TanStack Start and Nitro. Use:

- Install command: `npm ci`
- Build command: `npm run build`
- Output directory: leave blank

See `DEPLOYMENT.md` for the full deployment checklist.
